## Descrição do Projeto

O sistema consiste em um gerenciador de alunos executado via terminal, permitindo o cadastro, consulta, atualização, remoção e organização de registros de alunos.

O projeto foi desenvolvido utilizando os principais conceitos da disciplina, como:

- Struct
- TAD (Tipo Abstrato de Dados)
- Modularização
- Memória dinâmica
- Manipulação de arquivos
- CRUD
- Ponteiros

---

## Funcionalidades

O sistema possui as seguintes funcionalidades:

1. Cadastrar aluno
2. Listar alunos
3. Buscar aluno por ID
4. Atualizar aluno
5. Remover aluno
6. Ordenar alunos por nome
7. Salvar dados automaticamente
8. Carregar dados automaticamente ao iniciar
